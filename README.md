# Shorti
A Shortener url gem for rails

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'shorti'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install shorti
```

Then install shorti:
```bash
$ rails g shorti:install
```
